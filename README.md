# DawnTranslator

Офлайн-переводчик англ → рус для Dawncaster (и других игр) на Android.

Работает как Accessibility Service: видит текст на экране → переводит → показывает в плавающем окне.

## Как получить APK (облачная сборка, GitHub Actions)

### Шаг 1: Залей проект на GitHub
1. Создай новый **приватный** репозиторий на github.com (или публичный, без разницы)
2. Скопируй все файлы из этой папки в репозиторий и сделай push

### Шаг 2: Запусти сборку
1. В репозитории зайди во вкладку **Actions**
2. Слева выбери **Build APK**
3. Нажми **Run workflow** → **Run workflow**
4. Подожди 3–4 минуты

### Шаг 3: Скачай APK
1. Когда сборка завершится (зелёная галочка), зайди в неё
2. Внизу будет **Artifacts** → **DawnTranslator-APK**
3. Скачай zip, внутри — `app-release.apk`
4. Закинь на планшет, установи

## Как пользоваться на планшете

1. **Открой DawnTranslator** — при первом запуске скачается модель перевода (~30 МБ)
2. **Дай разрешение на оверлей** — кнопка в приложении
3. **Включи Accessibility** — кнопка в приложении, найди DawnTranslator в списке и включи
4. **Открой Dawncaster** — перевод будет появляться автоматически

## Как дополнить словарь

Отредактируй файл `app/src/main/java/com/dawncaster/translator/Translator.kt`:
- Секция `exactGlossary` — точные совпадения фраз (например «Bleed» → «Кровоток»)
- Секция `partialGlossary` — замена слов внутри перевода (например «bleeding» → «кровоточащий»)

После правок — залей в репозиторий и пересобери через Actions.

## Требования
- Android 8.0+
- ~50 МБ свободного места (модель перевода)
- Интернет только при первом запуске (скачать модель)
